## 使用方法
1. [下载](https://github.com/bitdust/WamaCry/releases)并解压文件夹
2. 在你想破坏的文件上右键->打开方式->选择默认程序
3. 浏览到程序文件夹并设置 WamaCry.exe 为这个文件类型的默认打开程序
4. 一脸真诚的去向老板报告情况
